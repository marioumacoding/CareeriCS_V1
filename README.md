# CareeriCS

## Overview

CareeriCS is a web platform that helps computer science students and recent graduates transition from academic learning to real-world careers.
The platform bridges the gap between university education and industry expectations by guiding users through a structured journey from career exploration to job preparation.

---

## Features

* Career path recommendation through an interactive quiz
* Personalized learning roadmaps
* Skill assessment and progress tracking
* CV builder and resume enhancement
* AI-powered interview practice
* Job and internship aggregation

---

## ⚙️ Getting Started

### Backend Setup

```
dotnet restore
dotnet run
```

### Frontend Setup

```
npm install
ng serve
```

Open the application at:

```
http://localhost:4200
```
